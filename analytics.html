Page Tracking Tags

All pages must be categorized with category,sub category tags are optional. These tagged using meta tags and placed within the head section of the code.

Usage: 


<head> 
	<meta name="coop_section" content="category name">
	<meta name="coop_sub1" content="sub category 1 name"> 
	<meta name="coop_sub2" content="sub category 2 name"> 
	<meta name="coop_sub3" content="sub category 3 name"> 
</head>

Example for usage: 


<head> 
	<meta name="coop_section" content="insurance">
	<meta name="coop_sub1" content="life"> 
</head>

Sub category is currently limited to coop_sub1, coop_sub2 and coop_sub3.

Event Tracking

To simply tracking, data attributes are used to track events. Any element can be tagged with the event tracking attributes and the event will be captured on element click.


<a href="url" data-coop-event="coop-event" data-coop-category="event category" data-coop-action="event action" data-coop-label="event label" data-coop-value="0" >click me!</a>

data-coop-event must be set to "coop-event", data-coop-category and data-coop-action must be set with a value, data-coop-label and data-coop-value are optional.
